PrimaryCensored.PrimaryCensoredDist — Type
struct PrimaryCensoredDist{D1<:(Distributions.UnivariateDistribution), D2<:(Distributions.UnivariateDistribution)} <: Distributions.UnivariateDistribution{Distributions.ValueSupport}

Generic wrapper for a primary event censored distribution.

Contructors

  • PrimaryCensoredDist(; uncensored::D, censoring::D): Constructs a PrimaryCensoredDist distribution with the specified uncensored and censoring distributions.

Examples

using PrimaryCensored, Distributions

uncensored = Normal()
censoring = Uniform(0, 1)

d = primarycensored(uncensored, censoring)

rand(d)

Fields

  • uncensored::Distributions.UnivariateDistribution: The original (uncensored) distribution.

  • censoring::Distributions.UnivariateDistribution: The primary event censoring distribution.

PrimaryCensored.WithinIntervalCensored — Type
struct WithinIntervalCensored{D<:(Distributions.UnivariateDistribution), L<:Real, U<:Real} <: Distributions.UnivariateDistribution{Distributions.ValueSupport}

Implement a censoring function.

Takes a UnivariateDistribution and returns a WithinIntervalCensored object but depends only on the cdf method.

Arguments

  • dist the truncated primarycensored distribution
  • lower the lower bound of the interval
  • upper the upper bound of the interval

  • WithinIntervalCensored object
